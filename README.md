Этот проект представляет собой многопоточный TCP-сервер на C++ с использованием библиотеки Boost.Asio, реализующий хранилище ключ-значение (Key-Value Storage). 
Сервер поддерживает работу множества клиентов одновременно, предоставляя базовые операции работы с данными, а также административные функции.

Базовые операции KV-хранилища:

GET <key> — получить значение по ключу

SET <key> <value> — установить значение для ключа

DEL <key> — удалить ключ

Административные команды:

CLIENTS — просмотр списка подключённых клиентов

LOG [N] — просмотр последних N строк лога (по умолчанию 30)

ADMIN <password> — повышение клиента до администратора

Технические особенности:

Асинхронная обработка подключений

Потокобезопасное KV-хранилище с использованием shared_mutex

Отдельно реализован клиентский интерфейс для удобного подключения к серверу с одного ПК

Cохранение бд на диск

К проекту прилагается CMakeLists.txt для простой компиляции.
Логирование операций с уровнями: DEBUG, INFO, WARNING, ERROR

Разграничение прав доступа (обычный пользователь / администратор)
